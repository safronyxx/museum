Опыт работы с технологиями

В ходе выполнения курсовой работы были использованы следующие технологии: Java 21, Spring Boot, Spring Data JPA (с Hibernate в качестве ORM), Spring Security для реализации ролевой модели и аутентификации, Thymeleaf — для построения динамического веб-интерфейса, PostgreSQL — в качестве системы управления базой данных. Проект разрабатывался в среде IntelliJ IDEA, сборка осуществлялась с помощью Maven. Была реализована трёхзвенная архитектура приложения (представление — сервер — данные), а также обеспечена защита от SQL-инъекций и XSS-уязвимостей за счёт использования ORM и шаблонизатора.
